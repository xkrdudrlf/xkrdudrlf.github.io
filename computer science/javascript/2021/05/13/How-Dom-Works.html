What is DOM ?

DOM is a structured representation of HTML documents. It allows JavaScript to access the HTML elements and styles to manipulate them.

DOCUMENT is the special object that is the entry point to the DOM like ‚Äúdocument.querySelector()‚Äù. And the child of DOM object is the <html> which is the root element of the all HTML elements. Next are the <head> and <body> elements which are the children of the <html> element.


DOM !== JavaScript

DOM methods and properties are not part of JavaScript. DOM methods and properties are actually part of WEB APIs which are like libraries that browsers implement in JavaScript language and we can access them from our JavaScript code. Since there is actually an official DOM specification each browser has to implement, we can do the same DOM manipulations in all browsers.
